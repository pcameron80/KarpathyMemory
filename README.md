# KarpathyMemory

Unified, cross-machine "second brain + Claude Code memory" system inspired by [Andrej Karpathy's LLM-knowledge-base tweet](https://x.com/karpathy/status/2039805659525644595).

Two Obsidian vaults (personal + work/ministry) synced across Macs via self-hosted CouchDB on the home NAS. Claude Code hooks build per-project knowledge bases automatically from session transcripts.
